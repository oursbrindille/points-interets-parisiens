# README

Plusieurs éléments :
- jupyter notebook à la racine "monuments-paris.ipynb" et "rois-france.ipynb"
- Dossier flaskApp pour le backend (en Flask)
- Dossier kings pour le frontend (en ReactJS)

## Mise en route :

1. cd flaskApp
2. python demo.py
3. cd ../kings
4. npm start
5. se rendre sur http://localhost:3000/

## API
